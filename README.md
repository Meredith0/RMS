* Spring Boot
* Spring Security
* Mybatis

前端
* Vue.js
* Element-UI

## Getting start

#### 前端初始化
```
cd ./vue-rms
#安装依赖
npm install
#运行, 在8080端口
npm run dev
```
#### 导入数据库

RMS/sql/database.sql

RMS/sql/data.sql

#### 运行后端
默认在8088端口

浏览器访问 http://localhost:8080
